# VDW_R0

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_R0 = [real (array)]

Description: VDW\_R0 defines the $R\_0$ parameters ($\AA$) for each species defined in the POSCAR file.

---

VDW\_R0 sets the $R\_0$ parameters ($\AA$) for each species defined in the POSCAR file for the DFT-D2, DFT-ulg, Tkatchenko-Scheffler, and Many-body dispersion energy methods.
