# CPARAM

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

CPARAM = [real]  
 Default: **CPARAM** = 0.0093

Description: The tag CPARAM allows to specify the value of the parameter $C$ in the kernel of the nonlocal rVV10 correlation functional.
