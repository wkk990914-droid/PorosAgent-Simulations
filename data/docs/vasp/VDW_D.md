# VDW_D

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_D = [real]

|  |  |  |
| --- | --- | --- |
| Default: **VDW\_D** | = 20 | for DFT-D2 and Tkatchenko-Scheffler |
|  | = 0.6966 | for DFT-ulg |
|  | = 6 | for Many-body dispersion energy |

Description: VDW\_D defines the damping parameter $d$.

---

VDW\_D sets the damping parameter $d$ for the DFT-D2, DFT-ulg, Tkatchenko-Scheffler, and Many-body dispersion energy methods.
