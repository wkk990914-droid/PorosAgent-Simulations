# LIBXC2_Pn

Categories: INCAR tag, Exchange-correlation functionals, Hybrid functionals

LIBXC2\_Pn = [real]

Description: LIBXC2\_Pn, where $n=1, 2, \ldots$ allows to specify the values of the parameters of the functional implemented in Libxc that is called with LIBXC2.

---

A detailed explanation on how to use LIBXC1\_Pn and LIBXC2\_Pn and an example can be found at the LIBXC1\_Pn page.
