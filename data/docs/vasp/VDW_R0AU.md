# VDW_R0AU

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_R0AU = [real (array)]

Description: VDW\_R0AU defines the $R\_0$ parameters (bohr) for each species defined in the POSCAR file.

---

VDW\_R0AU sets the $R\_0$ parameters (bohr) for each species defined in the POSCAR file for the Tkatchenko-Scheffler and Many-body dispersion energy methods.
