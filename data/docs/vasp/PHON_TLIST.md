# PHON_TLIST

Categories: INCAR tag, Electron-phonon interactions, Phonons

PHON\_TLIST = [real array]  
 Default: **PHON\_TLIST** = none

Description: Provides the list of temperatures for that the electron-phonon interaction using the ZG configuration is evaluated.

---

Also, the number of temperatures defined in PHON\_NTLIST has to be defined together with this tag.

For further usage of this tag see: Electron-phonon interactions from Monte-Carlo sampling.

> **Mind:** This feature is available for VASP >= 6.0.
