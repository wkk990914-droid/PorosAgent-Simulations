# NPPSTR

Categories: INCAR tag, Linear response, Dielectric properties, Berry phases

IGPAR = [integer]

Description: NPPSTR specifies the number of k-points on the strings in the IGPAR direction.

---

NPPSTR specifies the number of k-points on the strings $\mathbf{k}\_{j} = \mathbf{k}\_{\perp} + j\mathbf{G}\_{\parallel}/\mathrm{NPPSTR}$ (with $j=0,..,\mathrm{NPPSTR}-1$). See also LBERRY.
