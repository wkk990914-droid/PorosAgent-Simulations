# OFIELD_A

Categories: INCAR tag, Advanced molecular-dynamics sampling

OFIELD\_A = [real]

Description: The tag OFIELD\_A sets the desired order parameter $Q\_6$ in the Interface pinning method.

---

The bias potential in the Interface pinning method is written as

:   :   $U\_\text{bias}(\mathbf{R}) = \frac\kappa2 \left(Q\_6(\mathbf{R}) - A\right)^2$.

The tag OFIELD\_A sets the desired value of the order parameter $A$.
