# GAMMA_VDW

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

GAMMA\_VDW = [real]

|  |  |  |
| --- | --- | --- |
| Default: **GAMMA\_VDW** | = 1.12 | for IVDW\_NL=3 or |
|  | = 1.29 | for IVDW\_NL=4 |

Description: Specify $\gamma$ in vdW-DF3-opt1/vdW-DF3-opt2.

---

The GAMMA\_VDW tag allows to specify the value of the parameter $\gamma$ in the kernel of the vdW-DF3-opt1/vdW-DF3-opt2 nonlocal van der Waals functionals.
