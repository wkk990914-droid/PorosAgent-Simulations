# VDW_S6

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_S6 = [real]

Description: VDW\_S6 defines the global scaling factor $s\_{6}$.

---

VDW\_S6 sets the global scaling factor $s\_{6}$ for the DFT-D2, DFT-D4, DFT-ulg, and Tkatchenko-Scheffler methods.
