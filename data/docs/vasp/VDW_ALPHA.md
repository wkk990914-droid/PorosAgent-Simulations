# VDW_ALPHA

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_ALPHA = [real (array)]

Description: VDW\_ALPHA defines the polarizability $\alpha\_{0}$ (bohr$^{3}$) for each species defined in the POSCAR file.

---

VDW\_ALPHA sets the polarizability $\alpha\_{0}$ (bohr$^{3}$) for each species defined in the POSCAR file for the Tkatchenko-Scheffler and Many-body dispersion energy methods.
