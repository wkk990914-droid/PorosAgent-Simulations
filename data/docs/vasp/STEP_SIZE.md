# STEP_SIZE

Categories: INCAR tag, Transition states, Molecules

STEP\_SIZE = [real]  
 Default: **STEP\_SIZE** = 0.01

Description: The flag STEP\_SIZE defines the trial step size for the optimization step (in $\AA$ ) in the Improved Dimer Method.
