# VDW_C6AU

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_C6AU = [real (array)]

Description: VDW\_C6AU defines the $C\_6$ parameters ($\mathrm{Hartree}$ $\mathrm{bohr}^{6}$) for each species defined in the POSCAR file.

---

VDW\_C6AU sets the $C\_6$ parameters ($\mathrm{Hartree}$ $\mathrm{bohr}^{6}$) for each species defined in the POSCAR file for the Tkatchenko-Scheffler and Many-body dispersion energy methods.
