# NMRCURBX

Categories: Files, Output files, NMR

```
   Warning: Not yet released!
```

This page contains information about a feature that will be available in a future
release of VASP. In other words, currently you cannot use it even with the latest version of VASP. The information may change significantly until it is released.

The NMRCURBX file contains the NMR current density in atomic units (hartree bohr$^2$). It is written if WRT\_NMRCUR is set. The format is the same as CHGCAR with a header to define the grid and then 3 blocks that correspond to $j\_x$, $j\_y$, and $j\_z$.
