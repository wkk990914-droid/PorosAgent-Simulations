# VDW_SR

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_SR = [real]

Description: VDW\_SR defines the damping function parameter $s\_{R}$ (or scaling factor).

---

VDW\_SR sets the damping function parameter $s\_{R}$ (or scaling factor) for the DFT-D2, DFT-D3, DDsC, Tkatchenko-Scheffler and Many-body dispersion energy methods.
