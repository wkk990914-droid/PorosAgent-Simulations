# REPORT

Categories: Files, Output files, Ensemble properties, Advanced molecular-dynamics sampling

The output file REPORT contains information about the MD run such as list of parameters used in the simulation, the values of controlled geometric parameters, number of collision with the heat-bath (Andersen thermostat), quantities needed to compute the free-energy gradient, etc.
