# VDW_A2

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_A2 = [real]

Description: VDW\_A2 defines the damping function parameter $a\_{2}$.

---

VDW\_A2 defines the damping function parameter $a\_{2}$ for the DFT-D3 (BJ damping function) and DFT-D4 methods.
