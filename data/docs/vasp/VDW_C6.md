# VDW_C6

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_C6 = [real (array)]

Description: VDW\_C6 defines the $C\_6$ parameters ($\mathrm{Jnm}^{6}\mathrm{mol}^{-1}$) for each species defined in the POSCAR file.

---

VDW\_C6 sets the $C\_6$ parameters ($\mathrm{Jnm}^{6}\mathrm{mol}^{-1}$) for each species defined in the POSCAR file for the DFT-D2, DFT-ulg, Tkatchenko-Scheffler, and Many-body dispersion energy methods.
