# VDW_A1

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

VDW\_A1 = [real]

|  |  |  |
| --- | --- | --- |
| Default: **VDW\_A1** | = 1.0 | for DFT-D3 with the BJ damping function |

Description: VDW\_A1 defines the damping function parameter $a\_{1}$.

---

VDW\_A1 sets the damping function parameter $a\_{1}$ for the DFT-D3 (BJ damping function) and DFT-D4 methods.
