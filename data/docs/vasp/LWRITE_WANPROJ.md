# LWRITE_WANPROJ

Categories: INCAR tag, Wannier functions, Constrained-random-phase approximation

LWRITE\_WANPROJ = [logical]  
 Default: **LWRITE\_WANPROJ** = .FALSE.

Description: LWRITE\_WANPROJ determines whether the Wannier projection file WANPROJ is written.

---

If WANPROJ is present, VASP skips the Wannier projection and reads the effective projectors Unk from the WANPROJ file for each k point.
