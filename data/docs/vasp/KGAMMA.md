# KGAMMA

Categories: INCAR tag, Crystal momentum

KGAMMA = [logical]  
 Default: **KGAMMA** = .TRUE.

Description: Determines whether the k points (specified by the KSPACING tag ) include (KGAMMA=.TRUE.) the $\Gamma$ point.

---

> **Important:** If a KPOINTS file is present, VASP ignores the KGAMMA tag and the KSPACING tag.
