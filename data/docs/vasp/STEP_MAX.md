# STEP_MAX

Categories: INCAR tag, Transition states, Molecules

STEP\_MAX = [real]  
 Default: **STEP\_MAX** = 0.1

Description: The flag STEP\_MAX defines the trust radius (upper limit) for the optimization step (in $\AA$ ) in the Improved Dimer Method.
