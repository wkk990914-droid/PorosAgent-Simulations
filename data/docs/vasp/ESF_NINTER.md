# ESF_NINTER

ESF\_NINTER = [integer]  
 Default: **ESF\_NINTER** = 15

Description: ESF\_NINTER sets the maximum number of interpolation steps for the electronic structure factor (ESF\_SPLINES).

---

If the threshold set by ESF\_CONV has not been reached within ESF\_NINTER iterations VASP will print a warning about insufficient convergence.
