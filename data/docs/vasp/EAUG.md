# EAUG

Categories: POTCAR tag, Projector-augmented-wave method

EAUG = [real]  
 Default: **EAUG** = value read from POTCAR

Description: EAUG specifies the energy cutoff for the plane-wave representation for the augmentation charges in eV for the pseudopotential it is read from.

---

For a multi-element POTCAR file, the maximum EAUG determines the cutoff energy for the plane-wave representation of the augmentation charges.

> **Deprecated:** The value of EAUG can be overwritten by setting ENAUG in the INCAR file, but this functionality is deprecated and should not be used anymore.
