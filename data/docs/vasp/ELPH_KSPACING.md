# ELPH KSPACING

Categories: INCAR tag, Electron-phonon interactions

ELPH\_KSPACING = [real]  
 Default: **ELPH\_KSPACING** = 0.5

Description: The tag ELPH\_KSPACING determines the **k** point mesh if the KPOINTS\_ELPH file
is not present.

> **Mind:** Available as of VASP 6.5.0

---

For additional details of how the **k** point mesh is generated, see the documentation of the KSPACING tag.
