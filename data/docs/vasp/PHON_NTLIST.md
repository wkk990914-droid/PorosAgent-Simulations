# PHON_NTLIST

Categories: INCAR tag, Electron-phonon interactions, Phonons

PHON\_NTLIST = [integer]  
 Default: **PHON\_NTLIST** = none

Description: Sets the number of temperatures for the stochastic approach to electron-phonon interactions.

---

PHON\_NTLIST corresponds to the number of entries read from the list of temperatures defined by PHON\_TLIST.
For a how-to on the stochastic approach to electron-phonon interactions see Electron-phonon interactions from Monte-Carlo sampling.

> **Mind:** This feature is available for VASP >= 6.0.
