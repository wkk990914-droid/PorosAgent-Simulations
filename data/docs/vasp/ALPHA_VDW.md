# ALPHA_VDW

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

ALPHA\_VDW = [real]

|  |  |  |
| --- | --- | --- |
| Default: **ALPHA\_VDW** | = 0.94950 | for IVDW\_NL=3 or |
|  | = 0.28248 | for IVDW\_NL=4 |

Description: Specify $\alpha$ in vdW-DF3-opt1/vdW-DF3-opt2.

---

The ALPHA\_VDW tag allows to specify the value of the parameter $\alpha$ in the kernel of the vdW-DF3-opt1/vdW-DF3-opt2 nonlocal van der Waals functionals.
