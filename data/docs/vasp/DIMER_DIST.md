# DIMER_DIST

Categories: INCAR tag, Transition states, Molecules

DIMER\_DIST = [real]  
 Default: **DIMER\_DIST** = 0.01

Description: The flag DIMER\_DIST defines the step size for the numerical differentiation (in $\AA$) for the Improved Dimer Method.
