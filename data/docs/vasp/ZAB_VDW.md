# ZAB_VDW

Categories: INCAR tag, Exchange-correlation functionals, Van der Waals functionals

ZAB\_VDW = [real]  
 Default: **ZAB\_VDW** = -0.8491

Description: The tag ZAB\_VDW specifies the value of the parameter $Z\_{ab}$ in the kernel of the nonlocal vdW-DF functional of Dion *et al*.

---

In a revised version of the vdW correlation functional from Lee *et al*., $Z\_{ab}=-1.8867$.
